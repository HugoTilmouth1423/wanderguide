## Pixel & Tracking Setup

**Events to track:**
1. `PageView` - Landing page hit
2. `ViewContent` - Scrolled to pricing
3. `InitiateCheckout` - Clicked "Get Started"
4. `StartTrial` - Asked first free question
5. `Purchase` - Bought any pass

**Custom Audiences to create:**
- Landing page visitors (for retargeting)
- Started but didn't buy (for retargeting)
- Purchasers (for lookalike)
